Distributed SLAM

Working on a distributed SLAM solution using multple drones for my Master's project. The basic platform is a Storm Q330 drone with an NVIDIA Jetson Nano attached for onboard compute. The idea is to have the Betaflight controller handle the low level flight controls while the Nano uses some type of neural network to support autonomous flight. The next step would be to implement a distributed SLAM solution on multiple drones.
